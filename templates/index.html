Инструкция по запуску на своей машине

  1. Скачать и установить Python 3.6 отсюда: https://www.continuum.io/downloads В процессе установки везде вводим "yes"
  2. Установить драйвера на видеокарту, как показано здесь: https://askubuntu.com/questions/851069/latest-nvidia-driver-on-ubuntu-16-04
  3. Открыть терминал. Super -> terminal. Ввести:
    conda install tensorflow-gpu
  4. Далее ввести:
    conda install keras
    conda install -c conda-forge opencv3
  5. Скачать и распаковать архив: source.tar.gz
  6. Зайти в папку. Кликнуть правой клавишей и выбрать:
    Open in Terminal. Ввести команду:
    python start.py
  7. В браузере перейти по адресу 127.0.0.1:5000
